## Install(安装)
1. 下载代码 | Download the code
```sh
git clone https://github.com/coder-syl/blog
```
2. 进入项目目录 | Go to the project catalog
```sh
cd blog
```

3. 安装依赖包 | Install the dependency package
```sh
yarn install
or
cnpm install
```

## Usage

本地启动应用 | Launch the app locally
```sh
yarn dev
or
cnpm run dev
```
